About This Project

Hello! My name is Nadaa Taiyab. I am a data scientist with a finance and health tech background. I started working on this project because my signficant other co-founded a company, Gruntwork, that is putting together a progressive equity scheme for their employees.

As the in-home Excel expert, I whipped up a spreadsheet to explore how different exit prices and equity allocations would impact payouts. Consequently, I got inspired to build an online tool that others could use to explore their own scenarios.

I also thought this would be an ideal project to practice Python and learn Flask, HTML, CSS, Docker, and AWS Elastic Beanstalk. The code is available on my github repo and you can pull the Docker image from Docker Hub.

This is my very first web app so please excuse some of the useabilty issues! I would love constructive feedback on the project and the code. Feel free to get in touch or submit a pull request if you would like to contribute to the project.
